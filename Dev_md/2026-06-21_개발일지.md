## 1. 날짜(Day)별 강의안 다운로드 + 미리보기 기능

- `src/lib/lectureExport.js` 신규 — `lectureNotes.js` 구조화 데이터를 **자체완결형 인쇄용 HTML**로 직렬화
  - `buildDayHtml(dayIdx)` : 표지(강사 정보 포함) + 교시별 전체 강의안(intro·학습목표·핵심개념·사례·슬라이드·실습·프롬프트·기술노트·요약·체크리스트·산출물·팁)
  - `downloadDayHtml(dayIdx)` : Blob → `조선대AI특강_DayN_강의안.html` 다운로드
  - 정적 사이트(GitHub Pages)라 서버 없이 미리보기·다운로드·PDF 인쇄 모두 클라이언트에서 동작
- `src/components/LecturePreviewModal.jsx` 신규 — iframe `srcDoc` 미리보기 모달
  - 액션: `🖨 인쇄·PDF 저장`(iframe.print) · `↓ HTML 다운로드` · `↓ PPT 다운로드`
  - ESC/배경 클릭 닫기, body 스크롤 잠금
- `LectureNotes.jsx` 사이드바: 기존 PPT 링크 → **`👁 미리보기·다운로드` 버튼 + `↓ PPT 다운로드`** 2버튼으로 분리

## 2. 새 PPT(2종) 점검 및 학습콘텐츠 정합

재업로드된 `chosun-ai-day1/2.pptx`(Day1 112슬라이드·Day2 77슬라이드) 텍스트를 추출·대조해 강의안을 PPT 기준으로 정렬.

- **강사 정보 명시**: `data.js` infoCards Faculty → 이애본 / aebon@hs.ac.kr / 한신대 AI.SW대학·드림아이티비즈, 다운로드 문서 표지에도 반영
- **교시별 실습 랩 프롬프트 교체(14교시)**: 기존 산문형 `promptExample` → PPT의 복사용 5요소 PROMPT(`[역할][맥락][과업][형식][제약]` 또는 작업 절차형)로 전면 교체
- **기술 노트 신설(14교시)**: PPT의 학술 심화(Day1 심화기법 / Day2 기술노트 — 구성적 정렬·Bloom·grounding/RAG·시스템 프롬프트 가드레일 등)를 `techNote` 필드로 추가 → 강의안·다운로드 양쪽 렌더
- **완료 체크리스트 신설(Day2 7교시)**: PPT 교시별 체크리스트를 `checklist` 필드로 추가
- **제목 교정**: d2-s5 `과목·연구 에이전트 구축` → `과목 에이전트 구축`(PPT 일치)
- `public/ppt/` 새 PPT 2종 교체(배포 시 다운로드본 갱신)

## 4. Day 1 부록(APPENDIX) 사이트 이식 — 활용 사례집

PPT Day1 부록(S69–S111)을 신규 페이지 **활용 사례집**(`#cases`)으로 이식.

- `src/appendixCases.js` 신규 — 8개 섹션·**35개 사례** + 프롬프트 테크닉 6가지
  - 분야별 대표 사례(6) · 조선대 단과대학별(인문사회5·자연공학IT4·의치약보건4·예술체육3·융합자율4·전문대학원5) · 업무 전반(강의설계·동료심사·코드통계·행정소통 4)
  - 각 사례: 상황 · 복사용 5요소 프롬프트 · 산출물 예시 · 검증 체크포인트
- `src/components/AppendixCases.jsx` 신규 — 좌측 섹션 내비 + 사례 카드(프롬프트 **복사 버튼**, 산출물, ✓ 체크 칩)
- `App.jsx` — `#cases` 라우트 + 상단 nav `활용 사례집` 추가
- `index.css` — 사례집 그리드 모바일 대응(.cases-grid/.cases-aside/.case-cards)
